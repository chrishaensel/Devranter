Display Devrant rants in your sidebar or content

== Description ==
Devranter (the UNOFFICIAL Devrant Wordpress plugin) will let you display random rants, the latest rant or a single rant on your wordpress blog either in the sidebar or in the content section.

== Installation ==
Simply download the plugin and install as you would with other plugins.
